NueroVista

NueroVista is a comprehensive mental health and wellness platform designed to provide accessible, user-friendly mental health support for teenagers and young adults. The platform combines traditional assessment tools with interactive features for an engaging experience.

TailwindCSS
GSAP Animations
PHP/MySQL
Web Audio API

Key features include interactive mental health assessments, therapeutic tools like Virtual Piano Therapy, gaming for stress relief, progress tracking with visual analytics, and a comprehensive resource center. The platform prioritizes accessibility, engagement, and privacy.
